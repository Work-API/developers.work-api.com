Overview

The Work API provides broad access to many of the work tools we all use on a day-to-day basis. In this guide we will walk through the process to get set up on the platform and make our first calls to the API.

The guide will initially only cover the process using the Work API Ruby SDK, but will be expanded later to cover other languages as we add support.

NOTE:
These guides, and the API itself, are in BETA and so under constant review. If you come across any mistakes or omissions, please create an issue on our GitHub repo and we'll get it fixed.
